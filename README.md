#Dynamic Time-of-Use Pricing Analysis: London SmartMeter Dataset

This repository contains a deep learning analysis of the London SmartMeter dataset to evaluate the efficiency of the Dynamic-Time-of-Use (DToU) pricing scheme in reducing overall energy consumption.

#Models
Three different deep learning models were used to predict and evaluate the efficiency of the DToU pricing scheme:
1. Prophet: To capture seasonality and daily trend analysis
2. XGBoost: For tabular feature regression
3. Long-Short-Term-Memory (LSTM): For time-series and temporal sequence analysis
